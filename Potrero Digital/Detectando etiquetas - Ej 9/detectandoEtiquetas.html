Detectando etiquetas
  1. < header> Aquí dentro se muestra una imagen con el logo de la empresa y una barra de navegación < /header>
  2. < nav> Aqui aparecen items que conducen a diferentes sectores de la pagina web < /nav>
  3. < div> Contiene 2 titulos de diferentes tamaños, parrafos, imagenes y un boton < /div>
  4. < h2> Podria ser tambien un h3, pero comparandolo con los diferentes titulos de la pagina web creo que es un h2 < /h2>
  5. < p> Parrafo que probablemente esté dentro de un div contenedor junto con la imagen del barco, el h3 "naval" y el botón "ver más" < /p>
  6. < h3> Titulo que probablemente esté dentro de un div contenedor junto con la imagen del elevador, un párrafo y el botón "ver más" < /h3>
  7. < button> Botón que probablemente conduce a otra html de la pagina web o despliega un contenido oculto< /button>
  8. < iframe>Contenido insertado, en este caso un video de YouTube < /iframe>
  9. < footer> Pie de la pagina, contiene diferentes etiquetas como para imagenes, listas, formularios, etc.< /footer>
  10. < img> Logotipo de la empresa. Esta etiqueta no se cierra
